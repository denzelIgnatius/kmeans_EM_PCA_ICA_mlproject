1. Please copy the notebook files and the dataset files to the same folder.
2. Open Anaconda Prompt and run the below commands:
	pip install tensorflow==1.15
	pip install keras
3. Open the notebooks using Jupyter notebooks.
4. Churn_Dataset.ipynb will contain all the executed code for the telecom customer churn dataset.
5. GPU_Dataset.ipynb will contain all the executed code for the GPU dataset.
